Support: https://discord.gg/abuse
Prefix: . | () = required, [] = optional

Information

help - view all of serenas cmds.
botinfo - shows info from serena.
invite - generates a invite for serena.
support - invites you to the support server.

Fun

google (search) - googles the provided search.
urban (term) - urban dictionarys the provided term.

Misc

servericon - shows the server icon.
serverbanner - shows server banner.
serverinfo - gets info on the server.
channelinfo - gets info on current channel.
avatar [@user|userID|blank] - shows member avatar.
userinfo [@user|userID|blank] - shows info on a user.

Moderation

unlock - unlocks the channel.
lock - locks the channel.
unmute (@user) - unmutes the pinged user.
mute (@user) [time] [reason] - mutes the pinged user.
kick (@user) [reason] - kicks the user.
unban (@user) - unbans a user.
ban (@user) [reason] - bans the pinged user.
purge (amount) - clears the set amount of messages.

Server

prefix (New_prefix) - changes the server prefix.
resetprefix - resets the changed prefix to default.
setme - sets up channel, roles & logs used for moderation.
setlogs (channelID) - sets a logging channel to moniter server & moderation changes.
